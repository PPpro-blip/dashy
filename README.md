# DashyCore

## AI Workspace for Everyone

DashyCore is a modern AI workspace designed for coding, chatting, creativity, voice interaction, image generation, music, and productivity—all in one place.

---

## Features

- 🧠 DASH-Complexity
- ⚡ DASH-AllRound
- 🚀 DASH-SuperFast
- 🎮 GameMode
- 🎙 Voice Input
- 🔊 Text-to-Speech
- 💬 Chat History
- 🖼 Image Generation
- 🎵 Lyrics Generator
- 🔥 Progressive Web App
- 🌙 Themes
- 🎨 Accent Colors
- 🔐 Firebase Authentication

---

## Tech Stack

- HTML
- CSS
- JavaScript
- Firebase Authentication
- Cloudflare Workers
- DeepSeek API
- OpenRouter
- Vercel

---

## Philosophy

DashyCore is built around capabilities—not providers.

Users interact with experiences such as:

- DASH-Complexity
- DASH-AllRound
- DASH-SuperFast
- GameMode

instead of choosing between AI vendors. The underlying provider may change over time, while the user experience remains consistent.
